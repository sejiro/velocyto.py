Welcome to velocyto.py!¶

Velocyto is a library for the analysis of RNA velocity.

Velocyto includes a command line tool and an analysis pipeline

Warning

This documentation page refers to an alpha version (0.13) of the software. API will be subject to changes until 1.0 release. Please report issues on our Github page.
